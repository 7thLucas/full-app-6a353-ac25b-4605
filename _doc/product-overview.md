# Product Overview — Tic Tac Toe

## Product Name
Tic Tac Toe

## Tagline
Permainan klasik, langsung di browser — tantang temanmu sekarang.

## What It Is
Aplikasi web permainan Tic Tac Toe sederhana berbasis browser. Pemain bermain di grid 3×3, bergantian menempatkan tanda X atau O, dengan tujuan mendapatkan tiga tanda berurutan (horizontal, vertikal, atau diagonal). Mendukung dua mode: dua pemain di perangkat yang sama (lokal) dan satu pemain melawan komputer (AI dasar).

## Target Users
- **Casual players**: siapa saja yang ingin bermain cepat tanpa install apapun
- **Teman & keluarga**: dua orang yang ingin bermain bersama di satu perangkat

## Core Features
1. **Grid 3×3 interaktif** — papan permainan klasik, klik sel untuk menempatkan tanda
2. **Mode 2 Pemain (Lokal)** — Player X vs Player O bergantian di satu perangkat
3. **Mode vs Komputer** — melawan AI sederhana (strategi dasar)
4. **Deteksi Menang / Seri** — sistem otomatis mendeteksi pemenang atau draw, menampilkan garis kemenangan
5. **Papan Skor** — melacak skor X, O, dan seri lintas ronde
6. **Mulai Ulang** — tombol reset untuk ronde baru dan reset skor total

## Brand & Tone
- **Visual**: bersih, minimalis, playful — warna cerah namun tidak berlebihan
- **Primary color**: Indigo (#6366f1)
- **Secondary color**: Amber (#f59e0b)
- **Tone**: ringan, menyenangkan, langsung — tidak formal

## Scope & Constraints
- Single-page web app (SPA feel), tidak perlu login atau akun
- Tidak ada multiplayer online / real-time
- No backend persistence — semua state di frontend (React state)
- Responsif untuk desktop dan tablet

## Strategic Principles
1. **Simplicity first**: satu halaman, tidak ada navigasi kompleks
2. **Instant play**: langsung bisa main tanpa setup
3. **Delight in details**: animasi ringan, feedback visual jelas saat menang/seri
